# Negocio

## Estructura principal

La experiencia `negocio` se divide en:

- `/negocio/inicio`
- `/negocio/escanear`
- `/negocio/gestionar`
- `/negocio/perfil`

## Inicio

La pantalla de inicio del negocio prioriza operacion:

- hero de negocio
- KPIs
- promos visibles
- beneficios
- estados vacios

Es la vista para entender actividad y rendimiento del negocio.

## Escanear

El negocio tambien usa una superficie de scanner.
Eso permite validar codigos o apoyar flujos ligados a canjes.

## Gestionar

`/negocio/gestionar` es la seccion mas administrativa del rol negocio.
Esta organizada por tabs internas:

- Seguridad
- Metricas
- Grupos
- Dispositivos

Desde aqui se concentran controles de operacion, monitoreo y organizacion.

## Perfil

El perfil negocio reutiliza el sistema compartido de perfil, pero con tarjetas de identidad y contenido especifico del negocio.

## Relacion con onboarding

El acceso real del negocio depende de:

- perfil del propietario
- datos del negocio
- categoria
- direccion
- verificacion de negocio
