# Auth y prototipos

## Auth

Las rutas `/` y `/auth` muestran un `AuthFlow` minimo.
Esa vista:

- pinta branding
- muestra el welcome step
- no conecta botones a acciones reales

## Paginas de prueba

Las rutas `PruebaPage`, `Prueba1Page`, `Prueba2Page`, `Prueba3Page` y `Prueba4Page` son pantallas de exploracion visual.

Lo que se ve en el repo:

- layouts de escritorio
- cards
- formularios estilizados
- previews de promo
- variantes de look and feel

## Crear promo

La ruta `/negocio/panel/crear-promo` monta una composicion visual amplia para el flujo de negocio.
Por su implementacion actual, se parece mas a un mock funcional que a una pantalla conectada a backend.

## Recomendacion de lectura

Usa esta app para entender propuestas visuales o interacciones exploratorias.
Para entender el producto operativo, usa `referidos-app`.
