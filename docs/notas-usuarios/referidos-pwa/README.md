# Referidos PWA experimental

## Que es

`referidos-pwa` no se comporta como la app principal.
Por su estructura actual, funciona como una superficie web ligera de pruebas visuales y auth simplificado.

## Rutas principales

- `/` y `/auth`
- `/prueba`
- `/prueba1`
- `/prueba2`
- `/prueba3`
- `/prueba4`
- `/negocio/panel/crear-promo`

## Resultado de la revision

- La entrada de auth es minima y solo renderiza una version estatica del welcome step.
- El peso real de esta app esta en las paginas `Prueba*` y en la maqueta de crear promo.
- Debe leerse como laboratorio o sandbox visual, no como reemplazo productivo de `referidos-app`.
