# Prelaunch

## Que es

`prelaunch` es la app publica de captacion.
Su objetivo no es operar el producto principal, sino explicar la propuesta, captar interesados y abrir canales de ayuda antes de entrar a la app principal.

## Superficies principales

- Landing de waitlist: `/`
- Landing comercial para negocios: `/negocios`
- Centro de ayuda general: `/ayuda/es`
- Centro de ayuda para negocios: `/ayuda-negocios/es`
- Soporte y feedback: `/soporte-chat`, `/soporte-correo`, `/feedback`
- Playground interno de componentes: `/components`

## Lo importante para un usuario

- Aqui se capta interes, no se administra la cuenta principal.
- Las rutas de ayuda y soporte sirven para resolver dudas antes del onboarding completo.
- La experiencia cambia entre desktop y mobile segun el dispositivo detectado.
