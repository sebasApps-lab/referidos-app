# Ayuda y soporte

## Centro de ayuda

Hay dos familias de ayuda:

- General: `/ayuda/es`
- Negocios: `/ayuda-negocios/es`

Cada una permite navegar:

- pagina raiz
- categorias
- articulos

Esto ordena el contenido antes de que el usuario tenga una cuenta plenamente activa.

## Canales de soporte

- `/soporte-chat`
- `/soporte-correo`
- `/feedback`

Estas rutas se apoyan en soporte anonimo.
Eso significa que una persona puede abrir un hilo sin iniciar sesion en la app principal.

## Que puede esperar el usuario

- Consulta de categorias antes de abrir un ticket
- Creacion de ticket anonimo
- Seguimiento del estado del hilo anonimo
- Cancelacion o retoma del hilo si el flujo lo permite

## Relacion con el producto principal

- Prelaunch sirve como antesala de soporte.
- Cuando el usuario ya entra a `referidos-app`, el soporte cambia a flujos autenticados y mas ricos.
